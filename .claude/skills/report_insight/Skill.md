# 直播业绩日报点评

读取主播业绩日报.md，分析数据后生成点评和建议，追加到报告末尾。

## 触发方式

- 用户命令："分析日报"、"添加点评"、"生成建议"
- 生成报告后主动询问："是否需要添加点评建议？"

## 分析框架

### 一、整体表现点评
- GMV环比变化（>5%优秀，<-5%需关注）
- ROI健康度（>=4优秀，3-4一般，<3偏低）
- 利润变化趋势
- 退货率水平（>15%偏高）

### 二、主播表现分析
- GMV最高主播（亮点）
- 利润为负的主播（风险点）
- ROI偏低的主播（<3）

### 三、时段洞察
- GMV最高时段及主播
- ROI最高时段及主播
- 转化率最高时段及主播
- ROI<3的时段（需优化）

### 四、行动建议
基于以上分析，给出2-4条可执行建议

## 输出格式

```markdown
---

## 五、点评与建议

### 整体表现
- [评价]

### 主播表现
- [亮点/问题]

### 时段洞察
- [发现]

### 行动建议
- [建议1]
- [建议2]
```

## 使用方式

```
用户: "分析日报数据"
→ 读取主播业绩日报.md
→ 分析各部分数据
→ 生成点评建议
→ 追加到文件
```
