# 推广研究 Skill

## Purpose

系统化地进行社交媒体推广机会研究，包括：
- 关键词搜索量与供需分析
- 竞品内容策略研究
- 用户画像与需求洞察
- 流量缺口与入场时机判断

## When to Use

Use this skill when:
- 推广新产品/服务到社交媒体
- 分析竞品推广策略
- 寻找流量洼地机会
- 评估市场入场可行性
- 制定内容推广计划

## Workflow

### Phase 1: 关键词机会识别

#### Step 1: 收集关键词数据
1. 确定核心产品/服务关键词
2. 使用平台指数工具获取搜索数据：
   - 抖音指数 (douyin.com)
   - 微信指数
   - 百度指数
3. 记录：搜索量、趋势、内容供给量

#### Step 2: 分析供需缺口
```
搜索指数高 + 内容供给低 = 高价值机会词
搜索指数高 + 内容供给高 = 竞争激烈
搜索指数低 = 无需求（放弃）
```

**参考数据格式：**
| 关键词 | 搜索指数 | 峰值 | 内容供给 | 机会等级 |
|--------|---------|------|---------|---------|
| token | 17.8万 | 95万 | 极低 | ⭐⭐⭐⭐⭐ |
| openclaw | 70万+ | - | 近乎为零 | ⭐⭐⭐⭐⭐ |

### Phase 2: 竞品内容分析

#### Step 1: 找到头部内容
1. 用关键词搜索平台内容
2. 筛选高赞/高播放视频
3. 记录视频URL、发布者、互动数据

#### Step 2: 深度分析高流量视频
使用 `video_analyzer` skill 获取：
- 视频标题、文案
- 评论区内容（用户需求）
- 发布时间与更新频率
- 互动率与转化路径

#### Step 3: 提炼竞品策略
分析维度：
- 内容形式（教程/测评/对比/故事）
- 钩子设计（前3秒如何抓眼球）
- 转化路径（主页链接/私信/评论区引导）
- 差异化定位（为什么选他）

### Phase 3: 用户画像研究

#### Step 1: 从评论区挖掘需求
批量获取竞品视频评论区：
```
高频问题 = 用户痛点
情绪词（急/求/坑）= 购买意愿强度
竞品吐槽 = 差异化机会
```

#### Step 2: 平台数据分析
从指数工具获取：
- 人群画像（性别/年龄/地域）
- 活跃时段
- 关联兴趣标签

### Phase 4: 机会评估与策略

#### Step 1: 判断入场时机
```
窗口期 = 搜索量上升 + 内容供给少 + 无头部玩家垄断
危险信号 = 大厂下场 + 内容同质化 + 政策风险
```

#### Step 2: 制定内容策略
基于研究输出：
1. 目标关键词列表（主词+长尾词）
2. 内容形式建议（对标+差异化）
3. 发布节奏规划
4. 转化路径设计

## Example Usage

**Example 1: Claude API 抖音推广研究**
- 关键词：token, openclaw, 大模型
- 发现：token搜索95万峰值，内容供给极低
- 结论：高价值机会词，建议入场

**Example 2: AI工具测评推广**
- 竞品：分析10个头部AI测评账号
- 发现：教程类内容互动率高，评论区求链接多
- 策略：制作"AI工具对比"系列，评论区引导私域

## Best Practices

1. **数据时效性**：平台趋势变化快，建议1-2周内完成研究并执行
2. **多平台验证**：抖音火的内容，小红书可能还没人做（跨平台机会）
3. **评论区是金矿**：比视频本身更能反映真实用户需求
4. **小步快跑**：不要等完美策略，先发3-5条测试反馈
