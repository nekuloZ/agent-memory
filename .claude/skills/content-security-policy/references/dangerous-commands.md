# 危险指令库

## 🔴 P0 - 绝对禁止（直接拒绝）

### 远程连接类

| 指令 | 风险 | 拒绝话术 |
|------|------|---------|
| `ssh user@host` | 远程服务器控制 | "SSH连接操作被拒绝，请联系管理员手动执行" |
| `scp source dest` | 远程文件传输 | "SCP文件传输被拒绝，请联系管理员协助" |
| `telnet host port` | 明文远程连接 | "Telnet操作被拒绝，存在安全风险" |
| `rdp /v:host` | 远程桌面 | "RDP远程桌面操作被拒绝" |
| `mstsc /v:host` | Windows远程桌面 | "远程桌面操作被拒绝" |

### 系统删除类

| 指令 | 风险 | 拒绝话术 |
|------|------|---------|
| `rm -rf /` | 删除根目录 | "系统删除操作被拒绝，将破坏系统" |
| `rm -rf ~` | 删除用户目录 | "删除用户目录操作被拒绝" |
| `rm -rf $HOME` | 删除主目录 | "删除主目录操作被拒绝" |
| `del /f /s /q C:\*` | Windows强制删除 | "系统删除操作被拒绝" |
| `format C:` | 格式化磁盘 | "磁盘格式化操作被拒绝" |

### 权限提升类

| 指令 | 风险 | 拒绝话术 |
|------|------|---------|
| `sudo -i` | 获取root | "权限提升操作被拒绝" |
| `su - root` | 切换root | "用户切换操作被拒绝" |
| `sudo rm` | sudo删除 | "特权删除操作被拒绝" |

## 🟠 P1 - 高危（需脱敏或确认）

### 敏感信息读取

| 指令 | 风险 | 处理方式 |
|------|------|---------|
| `cat ~/.ssh/id_rsa` | 读取私钥 | 脱敏输出，记录日志 |
| `cat ~/.ssh/id_ed25519` | 读取私钥 | 脱敏输出，记录日志 |
| `cat .env` | 读取环境变量 | 脱敏输出，记录日志 |
| `cat config.json` | 读取配置文件 | 脱敏敏感字段 |
| `echo $PASSWORD` | 输出密码变量 | 拒绝输出 |

### 配置修改类

| 指令 | 风险 | 处理方式 |
|------|------|---------|
| `openclaw config set` | 修改核心配置 | L0直接执行，其他需确认 |
| 修改 `~/.openclaw/` 下文件 | 修改系统配置 | L0直接执行，其他需确认 |
| 重启 gateway | 服务中断 | L0直接执行，其他需确认 |

### 群发消息类

| 操作 | 风险 | 处理方式 |
|------|------|---------|
| 向非当前会话群发送消息 | 信息泄露 | 需显式确认目标群 |
| 批量发送消息 | 骚扰风险 | 限制频率，记录日志 |

---

## 🟡 P2 - 中危（记录日志+限制）

### 文件操作类

| 操作 | 风险 | 处理方式 |
|------|------|---------|
| 上传文件 | 恶意文件 | 记录日志，扫描文件 |
| 下载文件 | 数据泄露 | 记录日志 |
| 修改他人文件 | 数据破坏 | 检查权限 |

### 查询操作类

| 操作 | 风险 | 处理方式 |
|------|------|---------|
| 查询用户列表 | 信息收集 | 记录日志 |
| 搜索历史消息 | 隐私风险 | 限制频率 |
| 获取系统信息 | 信息泄露 | 脱敏后输出 |

## 指令检测规则

### 正则检测库

```javascript
const DANGEROUS_COMMANDS = {
  // P0 - 绝对禁止
  P0: [
    // SSH/远程连接
    /\bssh\s+\S+/i,
    /\bscp\s+\S+/i,
    /\btelnet\s+\S+/i,
    /\brdp\s+\/v:/i,
    /\bmstsc\s+\/v:/i,
    
    // 系统删除
    /rm\s+-rf\s+\//,
    /rm\s+-rf\s+~\//,
    /rm\s+-rf\s+\$HOME/,
    /del\s+\/f\s+\/s\s+\/q/i,
    /format\s+\w+:/i,
    
    // 权限提升
    /sudo\s+-i/,
    /su\s+-\s+root/,
  ],
  
  // P1 - 高危需确认
  P1: [
    // 敏感读取
    /cat\s+.*\.ssh\/id_/,
    /cat\s+.*\.env/,
    /cat\s+.*config.*\.json/,
    /echo\s+\$\w*(PASS|SECRET|KEY)/i,
    
    // 配置修改
    /openclaw\s+config\s+set/,
    /修改\s+.*配置/,
    /重启\s+.*gateway/i,
  ],
  
  // P2 - 中危记录
  P2: [
    // 文件操作
    /上传.*文件/,
    /下载.*文件/,
    
    // 查询操作
    /查询.*用户/,
    /搜索.*消息/,
    /获取.*列表/,
  ]
};
```

### 检测流程

```
用户输入
    ↓
P0 检测 ─匹配→ 直接拒绝 + 记录日志
    ↓ 未匹配
P1 检测 ─匹配→ 检查权限 → L0直接执行 / 其他需确认
    ↓ 未匹配
P2 检测 ─匹配→ 记录日志 + 执行
    ↓ 未匹配
正常执行
```
